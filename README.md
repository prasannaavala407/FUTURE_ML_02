# Future Interns - Task 2

## Support Ticket Classification using NLP

### Task Description
Classify customer support tickets into categories using Machine Learning

### What I Did
- Created synthetic dataset with 400 support tickets
- Text preprocessing: Lowercase + Special character removal
- Used TF-IDF Vectorizer for text features
- Trained Multinomial Naive Bayes classifier
- Achieved 100% accuracy
- Visualized predicted ticket categories

### Tech Stack
Python, Pandas, Scikit-learn, Matplotlib, TF-IDF, Naive Bayes
